Tutorial: Importar productos de maquinaria en Odoo 17
━━ SOLAPA 1 — ALTA DE PRODUCTOS
1
Preparar el archivo del proveedor
El Excel debe tener estas columnas en orden: Proveedor · CUIT · Lista · Vigencia · Ref Interna · Titulo · Moneda · Alicuota IVA · Precio Costo · Precio Venta

2
Generar archivo 01 — Solapa 1 de este HTML
Arrastrá el archivo del proveedor → clic en Generar archivo 01 — Productos → descargá 01_Importar_Productos.xlsx.
3
Importar los productos en Odoo
Inventario → Productos → Acción → Importar registros
Subí 01_Importar_Productos.xlsx → Probar → si todo verde → Importar.
4
Exportar los productos recién creados
Inventario → Productos → filtrá por marca → seleccioná todos → Acción → Exportar
Buscá la plantilla "Para Importar Lista de Precios Nuevas HTML" → descargá el archivo.
━━ SOLAPA 2 — ACTIVIDAD AFIP + PRECIOS + PROVEEDOR
5
Generar archivos 02, 03 y 04 — Solapa 2 de este HTML
Subí el archivo del proveedor original (paso 1) + la exportación de Odoo (paso 4).
Verificá que la columna Match muestre ✅ en todos → clic en Generar archivos 02 · 03 · 04.
6
Importar Actividad AFIP
Inventario → Actividad AFIP Mapping → Acción → Importar registros
Subí 02_Importar_ActividadAFIP.xlsx → mapeá → Probar → Importar.
Primera vez: Ajustes → Técnico → Modelos → product.template.activity.mapping → Crear un menú → nombre Actividad AFIP Mapping → menú padre Inventario → F5
7
Importar Lista de Precios
Ventas → Items Lista Precios → Acción → Importar registros
Subí 03_Importar_ListaPrecios.xlsx → mapeá → Probar → Importar.
8
Importar Proveedor
Inventario → Proveedores Productos → Acción → Importar registros
Subí 04_Importar_Proveedor.xlsx → mapeá → Probar → Importar.
Notas importantes:
· El conversor detecta automáticamente IVA 10.5% o 21% y la moneda (ARS / USD / EUR / DHM).
· Precio de venta siempre $1 y costo $0 en el alta — los precios reales van en archivos 03 y 04.
· Si un producto ya existe, inactivalo antes de importar: abrí el producto → Acción → Archivar.
